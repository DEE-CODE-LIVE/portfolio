About Me

I’m a Data Engineer with 5 years of experience designing, building, and optimizing scalable, high-performance data platforms across finance, healthcare, insurance, and manufacturing. My passion is turning raw, complex datasets into valuable, actionable insights.

I specialize in cloud-native architectures on AWS (S3, EC2, EMR, Redshift, Lambda, Kinesis, SageMaker), Azure (Data Factory, Data Lake, Synapse, Databricks, Azure SQL, Azure DevOps), and GCP (BigQuery, Dataproc, Cloud Functions, Vertex AI). I’ve built both batch and real-time pipelines using Apache Spark (PySpark, SQL, MLlib), Kafka, Hadoop (Hive, Sqoop, HDFS), and orchestration tools like Airflow, AWS Glue, and Azure Data Factory.

My work covers the full data lifecycle: ingestion, transformation, storage, governance, and delivery. I’ve designed dimensional models (star/snowflake schemas), implemented Lakehouse architectures with Delta Lake and Iceberg, and integrated systems with CI/CD pipelines using Jenkins, Git, Docker, Kubernetes, and Terraform.

Beyond engineering, I create data visualization and monitoring solutions in Power BI, Tableau, QuickSight, Grafana, and Kibana empowering stakeholders to make fast, informed decisions.

My projects range from real-time fraud detection pipelines for insurance, to healthcare data platforms for clinical analytics, to predictive maintenance systems for manufacturing. I thrive in collaborative, cross-functional teams and am always learning new tools to stay ahead in the evolving data landscape.
